Find a model

Use guild search to find a model for your application. Guild package repositories contain dozens of state of the art models, many of which have been pretrained on large datasets, allowing you to fine tune them with your own data.

Train a model with your data

Train installed models using guild train. Guild will run and monitor the training operation, capturing TensorFlow summaries as well as system statistics like CPU and GPU performance metrics.

Visualize progress and results

Start Guild's advanced visualizer using guild view and view training progress and results. Guild provides real-time updates of metrics so you can keep up-to-date on your operations.

Select and deploy your model

If you're happy with your trained model, deploy it to an environment using guild deploy. Guild integrates with Google Cloud ML, Amazon EC2, and your data center on premises.

Guild AI works with your TensorFlow code

Train, evaluate, and serve your models using simple command line operations.

Command line interface

Guild reduces complex model operations to single commands. This makes your life easier and it also makes your models easier to use by others.

prepare
prepares a model for training (e.g. download and pre-process a dataset)
train
trains a model defined in your Guild project file
view
start Guild View - a browser interface for viewing training results for your project
evaluate
evaluate a trained model's performance
serve
serve a trained model locally over HTTP + JSON

Guild project file

To enable Guild features, add a Guild file to your project.

{% highlight none %}[project] name = MNIST [model] runtime = tensorflow train = intro prepare = intro --prepare evaluate = intro --evaluate [flags] batch-size = 100 epochs = 5{% endhighlight %}
